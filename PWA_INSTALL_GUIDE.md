## 📲 安装方法（安卓手机）

### 方法 1：Chrome 浏览器安装（推荐）

1. **打开 Chrome 浏览器**
   ```
   打开手机上的 Chrome 浏览器
   ```

2. **访问 Dashboard**
   ```
   在地址栏输入:
   https://crypto-dashboard.loca.lt/dashboard_pwa.html
   ```

3. **等待页面加载完成**
   ```
   页面完全加载后，等待几秒钟
   ```

4. **点击菜单按钮**
   ```
   点击 Chrome 右上角的三个点菜单
   ```

5. **选择"添加到主屏幕"**
   ```
   在菜单中找到并点击"添加到主屏幕"或"安装应用"
   ```

6. **确认安装**
   ```
   点击"添加"或"安装"按钮
   ```

7. **完成！**
   ```
   回到主屏幕，您会看到 Crypto Projects 的图标
   点击即可像原生 APP 一样使用！
   ```

## ✅ PWA 特性

安装到手机后，APP 具有以下特性：

- ✅ **离线访问**：部分功能可以在没有网络时使用
- ✅ **全屏显示**：像原生 APP 一样全屏运行
- ✅ **快捷方式**：在主屏幕有专属图标
- ✅ **推送通知**：可以接收推送通知（未来功能）
- ✅ **快速启动**：点击图标即可快速打开
- ✅ **原生体验**：接近原生 APP 的用户体验

## ⚠️ 注意事项

### 1. HTTPS 必需
PWA 需要 HTTPS 协议才能安装。
- ✅ 使用 https://crypto-dashboard.loca.lt
- ❌ 不要使用 http://

### 2. 浏览器要求
- Chrome 72+ (推荐)
- Firefox 60+
- Samsung Internet 7.2+

### 3. 首次安装
首次安装时可能需要：
- 允许浏览器"添加到主屏幕"权限
- 允许浏览器"安装应用"权限

### 4. 图标显示
如果图标没有显示：
- 清除浏览器缓存
- 重新安装 APP

## 🔧 故障排查

### 问题：找不到"添加到主屏幕"选项

**解决方案：**
1. 确保使用 Chrome 浏览器
2. 确保地址以 `https://` 开头
3. 刷新页面后重试
4. 清除浏览器缓存

### 问题：安装后图标不显示

**解决方案：**
1. 卸载 APP
2. 清除浏览器缓存
3. 重新安装

### 问题：APP 无法打开

**解决方案：**
1. 检查网络连接
2. 确保公网隧道正在运行
3. 尝试重新安装
